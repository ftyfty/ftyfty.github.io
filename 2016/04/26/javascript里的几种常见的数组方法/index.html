发表于 2016-04-26

Array()的几种方法

  1. splice(2,3,4)删除数组中任意项(三个参数).
  2. splice(1,3)删除从第一项开始的往后三项(两个参数).(splice可以结合pop(),unshift(),push(),shift()来理解)
  3. push()在数组的末尾增加一项.
  4. unshift()在数组的开头增加一项.
  1. pop()在数组的末尾删除一项.
  2. shift()在数组的开头删除一项.
  3. slice(3,5)复制数组(第一个参数代表从第几项开始复制,第二项代表复制的项数,项数不得低于第一个参数的值).
  4. join(‘+’)将数组中的每一项用指定的分隔符拼接成一个新的字符串.
  5. reverse()倒叙排列,原有的数组改变.
  6. indexOf()返回获取项在数组中出现的索引.
