HEXO+Github,搭建属于自己的博客

发表于 2015-07-02

hexo是一款基于Node.js的静态博客框架。

配置环境

安装node.js（node官网下载对应系统的版本，一路下一步安装即可。）
安装git（git官网下载对应系统的版本，安装即可。）
申请github账号（github）

安装HEXO

Node和Git都安装好后,首先创建一个文件夹,如blog,用户存放hexo的配置文件,然后进入blog里安装Hexo。
Shift+右击 打开命令窗口，执行如下命令安装Hexo：
npm install hexo-cli -g
执行init命令初始化hexo,命令：
hexo init blog
下载Hexo需要的文件：
npm install
启动Hexo:
hexo server

配置github

建立与你用户名对应的仓库，仓库名必须为【yourUserName(你的用户名).github.io】
部署到Github前需要配置_config.yml文件，拉到底部找到下面代码：
deploy:
type:
将它们修改为如下代码：
deploy:
type: git
repo: 你的仓库链接
branch: master
执行下面的命令，将markdown文件生成静态网页:
hexo generate
然后，执行配置命令上传到Github：
hexo deploy
然后再浏览器中输入https://ftyfty.github.io就行了，我的github的账户叫ftyfty,把这个改成你github的账户名就行了

总结

每次部署的步骤，可按以下三步来进行:
hexo clean
hexo generate
hexo deploy
本地调试(简写)
hexo g #生成
hexo s #启动本地服务，进行文章预览调试
常用命令
hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #将.deploy目录部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本
